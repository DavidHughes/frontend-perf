Drupal modules

Drupal offers a number of modules specifically designed to improve frontend performance. Combined with a well-made theme, these modules can help your Drupal sites fly!

Activities

How many of these goals can you complete?

CSS/JS Optimization

  1. Use the Magic module to move your scripts to the bottom.
  2. Use the Magic module to retain jQuery in the <head>.
  3. Use the AdvAgg module to generate compressed assets.
  4. Use the AdvAgg module to minify JavaScript.

Theming Optimization

  1. Use the Modernizr module to conditionally load a stylesheet (using .info file).
  2. Use the Modernizr API to conditionally load a polyfill (using template.php or a custom module).
  3. Add a DNS pre-fetch to your theme's page template
  4. Add a prerender to your theme's page template (must use Chrome or IE11 to test).

Media optimization

  1. Use the Picture module to add responsive images to your site.
  2. Use the Image Lazyloader module to defer images until they are actually seen.
  3. Use the CDN module to geographically distribute your site's assets.
